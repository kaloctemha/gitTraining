Git Cheat Sheet

Config
git config - configure git system wide, global or local
Create
git init - create empty git repo in current directory
Workflow
git add - add file to staging area
git commit - copy the snapshot from staged area to local repo
git reset - unstage a file
git restore - unstage a file or reset file in working directory
git checkout - reset file in working directory
Inspection & History
git status
git diff
git log
git show
git ls-files
File Operations
git clean
git rm
git mv
Branching
git stash
git branch
git checkout
git switch
git merge
